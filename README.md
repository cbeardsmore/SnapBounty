## Build and Run

- Install [Flutter](https://flutter.dev/docs/get-started/install) and confirm installation:
	
	```bash
	flutter doctor
	```

- Build the APK:

	```bash
	flutter build apk
	```

- Build the Android App Bundle

	```bash
	flutter build appbundle
	```

- Run the app:

	```
	flutter run
	```
